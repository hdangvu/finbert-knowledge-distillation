# Data directory

## Structure
- `data/raw/`: Full scraped dataset (not tracked in git).
- `data/processed/`: Cleaned/model-ready data (not tracked in git).
- `data/sample/`: Small sample metadata for demonstration (tracked in git).

## Note
This repository does not redistribute the full scraped article text publicly.
The `data/sample/` files contain only non-text metadata and derived statistics
to illustrate the schema and pipeline.
